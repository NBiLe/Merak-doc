一个事务包含多个操作。

操作类型如下表所示。

| **类型** | **编号** | **说明** |
| :--- | :--- | :--- |
| CREATE\_ACCOUNT | 0 | 创建一个新账户 |
| PAYMENT | 1 | 支付操作 |
| PATH\_PAYMENT | 2 | 路径支付操作 |
| MANAGE\_OFFER | 3 | 增加、删除和修改挂单 |
| CREATE\_PASSIVE\_OFFER | 4 | 创建反向挂单 |
| SET\_OPTIONS | 5 | 操作设置 |
| CHANGE\_TRUST | 6 | 增加、更新和删除信任线 |
| ALLOW\_TRUST | 7 | 允许信任 |
| ACCOUNT\_MERGE | 8 | 删除账户并转移账户余额到目标账户 |
| INFLATION | 9 | 运行通货膨胀 |
